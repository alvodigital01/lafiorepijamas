# La Fiore Pijamas

Site estático da La Fiore Pijamas.

## Estrutura

- `index.html`: página, estilos e scripts do site.
- `assets/images/branding`: logotipos e favicons.
- `assets/images/banners`: banners para desktop e mobile.
- `assets/images/institucional`: imagens institucionais e materiais antigos.
- `assets/images/produtos`: fotos organizadas por categoria.
- `assets/images/legado`: fotos não utilizadas atualmente pelo site.
- `assets/icons/pagamentos`: ícones das formas de pagamento.
- `assets/arquivos`: arquivos de origem e materiais auxiliares.
