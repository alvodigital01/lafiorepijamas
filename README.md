# La Fiore Pijamas

Site estático da La Fiore Pijamas.

## Estrutura

- `index.html`: página, estilos e scripts do site.
- `assets/images/branding`: logotipos e favicons em uso.
- `assets/images/banners`: banners para desktop e mobile em uso.
- `assets/images/institucional`: imagens institucionais em uso.
- `assets/images/produtos`: fotos de produtos em uso, organizadas por categoria.
- `assets/images/legado`: imagens não utilizadas atualmente pelo site (fotos antigas, variações de banner/logo não publicadas, sobras de catálogo), organizadas espelhando a mesma subpasta de origem.
- `assets/arquivos`: arquivos de origem e materiais auxiliares (não usados pelo site).
- `assets/_backup-originais`: cópias em `.jpg`/`.png` das imagens que hoje são servidas em `.webp`, mantidas apenas como backup rápido para reverter a conversão. Espelham o caminho original em `assets/images`.
- `assets/_nao-usados`: assets sem nenhuma referência no `index.html` no momento (ex.: ícones de pagamento de uma versão antiga do rodapé). Mantidos para eventual reaproveitamento futuro.

Todas as imagens ativas do site estão em formato `.webp` (ver `assets/_backup-originais` para os arquivos originais).
